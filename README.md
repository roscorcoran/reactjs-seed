# reactjs-seed
Example project using React(JSX) and Pure CSS

Built using gulp, babel, jade and less

Written in ECMAScript 6

## Installation

Install dependencies via npm & bower:

`npm install && bower install`

## Run

Run the application via gulp:

`gulp`
